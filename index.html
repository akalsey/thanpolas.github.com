Featured Work

Kickq

Kickq is a Job Queuing system for Node.js built on top of Redis. It features reliable and fast handling of Jobs, delayed execution and an option for the Job Creator to get notified on the spot if the Job completes.

Mantri

MantriJS is a Dependency Management System for medium to large web applications. It wraps around and exposes the powerfull Google Closure Tools, the same tools used by most of Google's web assets.

Assetflow

Assetflow is an asset deployment tool ala Rails Asset Pipeline. It supports md5 hash comparison with S3 and enables you to create powerful asset flows easily and fast. It is a Grunt task and applies solid cache-busting techniques transparently.

Grunt Closure Tools

Grunt Closure Tools for Grunt is a superset of the tools provided by the Google Closure Library. The powerful compiler, the builder and the dependency writer.
